## 🚀 Getting Started

### Prerequisites
- Python 3.10+
- Streamlit

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/ml-apps-hub.git
   cd ml-apps-hub
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the application:
   ```bash
   streamlit run nocodeds_index_app_v7.py
   ```
